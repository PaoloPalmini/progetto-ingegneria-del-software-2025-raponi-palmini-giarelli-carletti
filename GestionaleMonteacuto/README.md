# GestionaleMonteacuto

Progetto MVC+DAO di esempio.

Avvio rapido:

- Esegui il main come script:
  - `python GestionaleMonteacuto/GestionaleMonteacuto.py`

- Oppure esegui il runner del package:
  - `python -m GestionaleMonteacuto.app.runner`

Contribuire:
- Aggiungi le `view` create dai colleghi nella cartella `view`.
- Aggiungi modelli concreti in `model` e DAO specifici in `dao`.
